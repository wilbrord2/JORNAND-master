AUTODESK AUTOCAD FOR ARCHITECTURAL DESIGN

ABOUT THE COURSE

In this Learning AutoCAD training course, expert author will teach you everything you need to know to be able to create 2D drawings and 3D models using the latest version of AutoCAD. This course is designed even for the absolute beginner, meaning no previous experience with AutoCAD is required.

You will start by learning the basic operations of AutoCAD. From there, you will be taught about the drawing and drafting tools and complete 3D modelling Tools for architectural design. Once you have completed this computer-based training course, you will have learned everything you need to know to create your own 2D drawings and 3D models in AutoCAD.

1, 2D Drawing with AutoCAD (For Architectural design)

CONTENT:

  • Lecture1: Getting familiar with AutoCAD

    Introduction and getting started, getting overview of AutoCAD user interface, Navigating tools.

  • Lecture 2: Starting with Drawings

    Making lines and deal with regular and irregular shapes, and drawings creating.

  • Lecture 3: Working on modifying tools

    Move and Copy, Rotate, offset, Mirror and Scale, Trim, extend, Erase, Explode and overkill, stretch command, Break and Join, Fillet and chamfer, Lengthen and Reverse.

  • Lecture 4: Drawing tools

    Quick measure tool, finding area, Making Isometric drawing, and gradient: Assigning object colour and Line type, assigning line weight and transparency, checking object properties.

  • Lecture 5: Working on layers

    Creating and assigning layers, using layer properties manager, working with layer states.

  • Lecture 6: Dimensioning and annotations

    creating basic dimensions, modifying dimensions, creating dimension style.

  • Lecture 7: Working with texts and letters

    creating single line text, creating text style, creating multiline text, creating and formatting table.

  • Lecture 8: Working with blocks and groups

    Creating simple Blocks, Inserting Blocks, Modifying and recall blocks, Working with group manager.

  • Lecture 9: Layouts and publishing

    Introduction to paper space or layout, Page setup for layout, setting scale of objects in viewports.

  • Lecture 10: Sheet set

    Introduction to sheet set, creating views in model drawings, making title block and template, Publishing sheet set.

  • Lecture 11: Completion of project drawings works

    provide floor plans, Elevations, Sections, Site plans with all environmental features, and other details of architectural drawings.

  • Lecture 12: Review and handover.
